Internal Links

Previous: Link Text or Alt Attribute

Next: Link vs. Button

If you want to make a link go to another point on the same page there are two ways to accomplish it - one correct way and one deprecated way.

Correct Implementation

Principle: Using the ID attribute to make an internal link

Source Code

<a href="#services">View all of our services</a>
<p>...</p>
<h2 id="services">Our Service</h2>

Implementation

View all of our services

...

Our Services

Incorrect Implementation

Principle: Using the name attribute to make an internal link

The name attribute has been deprecated in HTML5 and should not be used.

Source Code

<a href="#services">View all of our services</a>
<p>...</p>
<a name="services"></a>
<h2>Our Service</h2>

Implementation

View all of our services

...

Our Services
