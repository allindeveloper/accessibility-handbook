The link text should not be too long

Previous: Using Descriptive Link Text

Next: Unique Link Text

Correct Implementation

Principle: The link text is short enough to be quickly read, descriptive enough to know where it will take you, and is found in a context that tells you want the link does.

<p>View examples, watch tutorials, download tools, and check your Web site's  accessibility at the <a href="http://accessibility.oit.ncsu.edu">NC  State IT Accessibility Web site</a></p>

Incorrect Implementation

Principle: The link is too long to be quickly read by screen readers.
